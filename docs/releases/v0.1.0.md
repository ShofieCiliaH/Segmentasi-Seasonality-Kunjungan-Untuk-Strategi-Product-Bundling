# Release Notes v0.1.0

Tanggal rilis: 2026-04-03

## Ringkasan

`v0.1.0` merupakan rilis awal repository untuk prototipe sistem segmentasi seasonality kunjungan wisata Padusan berbasis Agglomerative Hierarchical Clustering.

## Cakupan Fitur

- upload file transaksi `.xlsx` dan `.csv`
- preprocessing data transaksi
- kategorisasi layanan wisata
- pembentukan matriks harian
- evaluasi jumlah cluster menggunakan Silhouette Score
- visualisasi dendrogram
- profil cluster dan strategi product bundling
- riwayat analisis
- ekspor laporan PDF

## Tujuan Rilis

Rilis ini ditujukan untuk:

- mendokumentasikan baseline prototype skripsi
- menyediakan kode yang siap dipresentasikan di GitHub
- memisahkan source code publik dari data operasional asli

## Catatan

- data operasional perusahaan tidak disertakan dalam release
- persistence pada prototype publik memakai JSON lokal
- struktur repo telah dirapikan agar siap untuk pengembangan lanjutan
